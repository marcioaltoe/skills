# Template

Substitua este corpo pelo conteúdo da sua skill. Mantenha a estrutura abaixo como guia.

## Quando usar

Descreva os gatilhos concretos que devem ativar essa skill. Exemplos:

- Usuário pede "X"
- Após uma sequência de Y
- Quando aparece o erro Z

## Como aplicar

Passos em imperativo. Seja específico — o agent segue literalmente.

1. Primeiro passo.
2. Segundo passo.
3. Terceiro passo.

## Anti-padrões

- O que NÃO fazer.
- Erros comuns que essa skill deve evitar.

## Referências (opcional)

- Links para material em `references/` ou docs externas.
