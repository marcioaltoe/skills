# LLM Wiki Tools

Tooling companion skills for LLM Wiki work. Install this set with `llm-wiki`, `write-common`, and `base-mcps` when maintaining an Obsidian or Markdown-based wiki.

## Install

```bash
bunx skills add marcioaltoe/skills/skills/llm-wiki-tools -g
```

Use a local path while developing:

```bash
bunx skills add ./skills/llm-wiki-tools --list
```

## Skills

| Skill               | Use for                                         |
| ------------------- | ----------------------------------------------- |
| `qmd`               | Searching Markdown knowledge bases.             |
| `obsidian-markdown` | Writing Obsidian-compatible Markdown notes.     |
| `obsidian-cli`      | Managing Obsidian vaults from the command line. |
| `obsidian-bases`    | Creating Obsidian Bases views and filters.      |
| `mermaid-syntax`    | Creating Mermaid diagrams in wiki notes.        |
