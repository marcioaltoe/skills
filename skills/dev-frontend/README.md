# Dev Frontend

Frontend development skills for React, UI systems, routing, styling, accessibility, Core Web Vitals, browser validation, and frontend documentation.

## Install

```bash
bunx skills add marcioaltoe/skills/skills/dev-frontend -g
```

Use a local path while developing:

```bash
bunx skills add ./skills/dev-frontend --list
```

## Use With

- `dev-core` for shared coding workflow and verification.
- `dev-methods` for architecture, planning, QA, and review.
- `writing` for frontend documentation.
