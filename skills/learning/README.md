# Learning

Skills for deliberate practice, learning plans, and skill development during AI-assisted work. Install this set when a user wants to turn code changes, tutorials, articles, or transcripts into active learning reps.

## Install

```bash
bunx skills add marcioaltoe/skills/skills/learning -g
```

Use a local path while developing:

```bash
bunx skills add ./skills/learning --list
```

## Skills

| Skill                    | Use for                                       |
| ------------------------ | --------------------------------------------- |
| `learning-opportunities` | Short learning exercises during coding work.  |
| `ship-learn-next`        | Turning learning content into shippable reps. |
