# Skill Authoring

Skills for creating, evaluating, packaging, and improving agent skills: structured creation workflows, authoring best practices, discovery, and autonomous optimization loops (`autoresearch`).

## Install

```bash
bunx skills add marcioaltoe/skills/skills/skill-authoring -g
```

Use a local path while developing:

```bash
bunx skills add ./skills/skill-authoring --list
```

## Use With

- `writing` for clear skill bodies, descriptions, and README prose.
- `research-tools` for sourcing external material a new skill needs.
