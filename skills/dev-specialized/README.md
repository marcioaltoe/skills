# Dev Specialized

Specialized development skills for AI SDKs, agent frameworks, jobs, observability, payments, messaging, and external APIs.

## Install

```bash
bunx skills add marcioaltoe/skills/skills/dev-specialized -g
```

Use a local path while developing:

```bash
bunx skills add ./skills/dev-specialized --list
```

## Use With

- `dev-core` for shared coding workflow and verification.
- `dev-backend` when specialized services are part of backend implementation.
- `dev-methods` for architecture, security, QA, and decomposition.
