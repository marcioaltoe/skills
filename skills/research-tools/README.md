# Research Tools

Shared MCP-backed skills for source discovery, web search, scraping, crawling, and research workflows. Install this set when a project context needs external sources or current web evidence.

## Install

```bash
bunx skills add marcioaltoe/skills/skills/research-tools -g
```

Use a local path while developing:

```bash
bunx skills add ./skills/research-tools --list
```

## Skills

| Skill            | Use for                                               |
| ---------------- | ----------------------------------------------------- |
| `exa-web-search` | Deep research, lead generation, and synthesis.        |
| `firecrawl`      | Web scraping, crawling, extraction, and page capture. |
