## How to use this reference

Every quoted claim or pattern recommendation in this skill traces back to one of the URLs below. Use this file to:

- Verify a claim before adopting it in code review.
- Trace an unfamiliar recommendation back to its primary source.
- Spot-check whether a citation is current (some posts get rewritten yearly).
- Find related reading when extending the skill.

Sources are grouped by axis. A handful appear in multiple axes — those are listed once in the most relevant axis and noted in _Cross-axis_ at the bottom.
