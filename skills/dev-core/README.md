# Dev Core

Core development workflow skills for TypeScript projects and general coding sessions. Install this set first for git, structured bug diagnosis, verification, testing, documentation hygiene, codebase orientation, and agent handoff workflows.

## Install

```bash
bunx skills add marcioaltoe/skills/skills/dev-core -g
```

Use a local path while developing:

```bash
bunx skills add ./skills/dev-core --list
```

## Use With

- `dev-frontend` for React and UI work.
- `dev-backend` for APIs, auth, data, and platform work.
- `dev-methods` for architecture, planning, QA, security, and analysis.
- `writing` for documentation and communication.
