# Dev Backend

Backend development skills for APIs, authentication, databases, migrations, and platform services.

## Install

```bash
bunx skills add marcioaltoe/skills/skills/dev-backend -g
```

Use a local path while developing:

```bash
bunx skills add ./skills/dev-backend --list
```

## Use With

- `dev-core` for shared coding workflow and verification.
- `dev-methods` for architecture, security, QA, and decomposition.
- `dev-specialized` for AI SDKs, agent frameworks, jobs, observability, payments, and external APIs.
- `writing` for backend documentation.
